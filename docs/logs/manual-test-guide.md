# 手动测试指南

## 测试准备

### 控制说明
- **方向键**: 移动
- **Z**: A 按钮（跳跃/确认）
- **X**: B 按钮（射击/取消）
- **Enter**: Start（开始）
- **RShift**: Select（选择）
- **Ctrl+C**: 退出

### 测试记录要点
1. 启动时间
2. 标题画面是否正常
3. 能否进入游戏
4. 角色是否可控
5. FPS 估算（观察流畅度）
6. 发现的问题
