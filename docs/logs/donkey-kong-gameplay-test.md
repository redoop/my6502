# Donkey Kong 游戏功能测试

**日期**: 2025-11-29  
**当前状态**: 有图形显示，颜色正常

---

## 🎮 测试步骤

### 1. 启动游戏
```bash
./scripts/run.sh games/Donkey-Kong.nes
```

### 2. 观察启动画面 (0-10秒)

**检查项**:
- [ ] 能看到 Nintendo 标志？
- [ ] 能看到 "DONKEY KONG" 标题？
- [ ] 画面是否稳定（不闪烁）？
- [ ] 颜色是否正常？

**记录**:
- 启动后多久出现标题: _____ 秒
- 画面描述: _____________________

---

### 3. 测试按键响应

**控制说明**:
- **Enter**: Start（开始游戏）
- **RShift**: Select（选择）
- **方向键**: 移动
- **Z**: A 按钮（跳跃）
- **X**: B 按钮

**测试**:
- [ ] 按 Enter 键，游戏是否有反应？
- [ ] 画面是否切换到游戏场景？
- [ ] 能看到马里奥（角色）吗？
- [ ] 能看到 Donkey Kong（大猩猩）吗？

---

### 4. 测试角色移动

**测试项**:
- [ ] 按 ← 左键，马里奥向左移动
- [ ] 按 → 右键，马里奥向右移动
- [ ] 按 Z 键，马里奥跳跃
- [ ] 按 ↑ 键，马里奥爬梯子（如果在梯子旁）

**记录**:
- 移动是否流畅: _____
- 是否有延迟: _____
- 动画是否正常: _____

---

### 5. 测试游戏元素

**检查项**:
- [ ] 能看到平台/楼梯？
- [ ] 能看到桶（障碍物）？
- [ ] 能看到梯子？
- [ ] 能看到公主？
- [ ] 分数显示正常？

---

### 6. 测试碰撞检测

**测试**:
- [ ] 马里奥碰到桶会死亡/受伤？
- [ ] 马里奥站在平台上不会掉下去？
- [ ] 马里奥可以爬梯子？
- [ ] 马里奥跳跃可以越过桶？

---
